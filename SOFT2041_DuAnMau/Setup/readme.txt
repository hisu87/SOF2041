-- Guide
	1. Run the SQL file "EDUSYS_SQL.sql" to create the SQLServer database
	2. Run the EDUSYS_SETUP.exe 

-- Requirements
	- Java 8 and forward (Beta version accepted)
	- SQL Server installed

-- Tool used
	- JDK 16.0 Oracle
	- JDBC 9.4
	- Netbean 12.4
	- Exe4j
	- Inno Setup
